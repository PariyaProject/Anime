# 次元城自动播放脚本 - 最终版本

## 📁 文件说明

### 🎯 要使用的脚本文件
1. **`main-script.user.js`** - 主页面脚本 (安装在 cycani.org)
2. **`player-script.user.js`** - 播放器脚本 v14.2.0 (安装在 player.cycanime.com)
   - ✅ 修复了跨域localStorage问题
   - ✅ 使用URL参数传递播放信息
   - ✅ 优化弹窗体验（自动消失、淡出效果）
   - ✅ **简化稳定的自动播放**（直接静音播放）
   - ✅ **智能声音恢复**（自动尝试 + 用户提示）
   - ✅ **增强视频检测**（支持iframe和多种播放器）
   - ✅ 完善的错误处理和调试信息

### 🚀 使用方法
1. 在 Tampermonkey 中创建两个脚本
2. 分别导入这两个文件
3. 设置主页面脚本匹配 `https://www.cycani.org/watch/*`
4. 设置播放器脚本匹配 `https://player.cycanime.com/*`

### ✅ 功能特性
- ✅ 自动播放视频
- ✅ 自动连播下一集
- ✅ 智能下一集检测
- ✅ 进度显示和控制
- ✅ 用户友好的界面

### 🔧 技术架构
- 双脚本配合：主页面负责检测和导航，播放器负责自动播放和连播
- **使用URL参数传递播放信息** (解决跨域localStorage问题)
- 通过 GM_xmlhttpRequest 避免跨域问题
- 基于真实浏览器调试测试优化

### 📖 详细说明
- 主页面脚本会在视频页面检测下一集信息
- **通过URL参数将播放信息传递给播放器页面** (解决跨域限制)
- 播放器脚本从URL参数读取数据并实现自动播放和进度监控
- 视频结束前30秒显示下一集预告，自动跳转

**这是最终的完整解决方案，已彻底解决跨域数据传递问题并经过实际测试验证。**
