# Cycani 视频代理服务 MVP

这是一个用于次元城(cycani.org)的视频代理服务，解决跨域问题并提供更好的播放体验。

注意：当前仓库的实际后端目录是 `backend/`，请按当前根目录工作区结构使用。

## 功能特性

- ✅ **跨域代理**: 绕过浏览器同源策略限制
- ✅ **自动播放**: 支持视频自动播放和下一集切换
- ✅ **现代播放器**: 基于 Plyr 的现代化播放器界面
- ✅ **调试信息**: 实时显示请求和响应信息
- ✅ **响应式设计**: 支持桌面和移动设备

## 快速开始

### 1. 安装依赖

```bash
cd backend
npm install
```

### 2. 启动服务

```bash
# 开发模式
npm run dev

# 生产模式
npm start
```

### 3. 访问应用

打开浏览器访问: http://localhost:3000
后端 API 默认运行在: http://localhost:3006

## API 接口

### 获取剧集信息
```
GET /api/episode/:bangumiId/:season/:episode
```

**参数:**
- `bangumiId`: 动画ID (例如: 3944)
- `season`: 季度 (例如: 1)
- `episode`: 集数 (例如: 14)

**响应示例:**
```json
{
  "success": true,
  "data": {
    "bangumiId": 3944,
    "season": 1,
    "episode": 14,
    "title": "剧集标题",
    "videoUrl": "视频地址",
    "originalUrl": "原始页面URL"
  }
}
```

### 视频代理
```
GET /api/video-proxy?url=VIDEO_URL
```

**参数:**
- `url`: 视频URL (需要URL编码)

### 视频流代理
```
GET /api/stream?url=VIDEO_URL
```

**参数:**
- `url`: 视频URL (需要URL编码)
- 支持 Range 请求用于断点续传

## 使用说明

### 1. 加载视频
1. 在输入框中填写动画ID、季度和集数
2. 点击"加载视频"按钮
3. 系统会自动解析并加载视频

### 2. 播放控制
- **播放/暂停**: 点击播放器按钮或按空格键
- **音量控制**: 使用播放器音量滑块
- **全屏**: 点击全屏按钮
- **下一集**: 点击"下一集"按钮或按 Ctrl+→

### 3. 自动播放
- 点击"自动播放"按钮开启/关闭
- 开启后，当前视频结束会自动加载下一集

### 4. 调试信息
- 底部调试面板显示实时日志
- 包含成功/错误/警告等不同级别的信息
- 帮助排查问题

## 技术栈

**后端:**
- Node.js + Express
- Axios (HTTP 请求)
- Cheerio (HTML 解析)
- Helmet (安全中间件)

**前端:**
- HTML5 + CSS3 + JavaScript
- Bootstrap 5 (UI 框架)
- Plyr.io (视频播放器)

## 开发说明

## 故障排除

### 常见问题

1. **服务器连接失败**
   - 检查 Node.js 服务是否启动
   - 确认端口 3000 未被占用

2. **视频加载失败**
   - 检查网络连接
   - 查看调试信息中的错误详情
   - 可能需要更新视频解析逻辑

3. **播放器无法播放**
   - 检查视频格式是否支持
   - 确认浏览器支持 HTML5 视频

### 获取帮助
- 查看浏览器控制台错误信息
- 检查 Express 服务器日志
- 确认所有依赖包正确安装
